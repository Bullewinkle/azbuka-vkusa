Установка шрифтов PostScript Type 1.

Шрифт формата PostScript (Type 1) поставляется в виде набора из 2-х файлов 
с расширениями PFB и PFM или 3-х файлов с расширениями PFB, AFM и INF. 
В последнем случае файл PFM будет создан системой в процессе установки.

Windows МЕ/95/98/NT

Если Вы работаете с Windows ME/98/95/NT, для установки PostScript Type 1 шрифтов, 
необходимо использовать утилиту АТМ (Adobe Type Manager). Вы можете бесплатно 
получить на Веб сайте компании Adobe легкую версию ATM --  ATM Light for Windows, 
http://www.adobe.com/products/atmlight/main.htm

Установка шрифтов в Windows МЕ/95/98/NT с помощью ATM.

1. Запускаем ATM. Start > Programs > Adobe > Adobe Type Manager
2. Выбираем закладку Fonts.
3. В окне <Source> выбираем Browse for Fonts.
4. Выбираем диск и папку, содержащие устанавливаемые шрифты. АТМ представит 
    перечень существующих шрифтов. Если список шрифтов не появился там, где вы 
   ожидаете, убедитесь, что шрифты находятся в разархивированном виде!
5. Если вы устанавливаете шрифты с дискеты или CD, убедитесь что напротив надписи 
   "Add without copying files" отсутствует метка (<галочка>).
6. Выбираем шрифты. Чтобы выбрать более одного шрифта, нажмите клавишу CTRL и, 
     удерживая её, кликните на каждый требуемый шрифт.
7. Чтобы добавить шрифты, кликните Add.
После установки может потребоваться перезагрузка прикладной программы или повторный 
выбор текущего принтера для того, чтобы вновь установленные шрифты появились в списке.

Windows XP or 2000

Если Вы работаете с операционными системами Windows XP или Windows 2000, нет 
необходимости в установке АТМ, так как он уже является частью этих систем.

1. Открываем Панель Управления. Выбираем Start > Settings> Control Panel 
   (или Start > Control Panel для XP).
2. Двойным нажатием левой кнопки мыши открываем папку Fonts.
3. В меню выбираем File> Install New Font.
4. Выберите дисковод и папку, содержащую устанавливаемые шрифты.
     Если список шрифтов не появился там, где вы ожидаете, убедитесь, что шрифты 
     находятся в разархивированном виде!
5. Выберите устанавливаемые шрифты. Чтобы выбрать более чем один шрифт, нажмите 
    клавишу CTRL и, удерживая ее, кликните на каждый требуемый шрифт.
6. Если вы устанавливаете шрифты с дискеты или CD, убедитесь что напротив надписи 
  "Copy fonts to Fonts folder" проставлена метка (<галочка>).
7. Кликните ОК, чтобы установить шрифты. 
После установки может потребоваться перезагрузка прикладной программы или повторный 
выбор текущего принтера для того, чтобы вновь установленные шрифты появились в списке.
